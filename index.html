unbuntu系统安装spark

By Sun Qian

发表于 2015-07-14 1:28

一.安装java环境

$ sudo apt-add-repository ppa:webupd8team/java
$ sudo apt-get update
$ sudo apt-get install oracle-java7-installer

检查是否安装成功：
$ java -version

成功的话将会显示
java version “1.7.072” Java(TM) SE Runtime Environment (build 1.7.072-b14) Java HotSpot(TM) 64-Bit Server VM (build 24.72-b04, mixed mode)

二.安装scala

从网上下载安装包，可以直接从官网下载或从终端输入下列代码
$ wget http://www.scala-lang.org/files/archive/scala-2.11.6.tgz
$ sudo tar xvf scala-2.11.6tgz -C /opt #将scala压缩至 /opt/scala-2.11.6
设置环境变量
$ gedit /etc/profile
在打开的文件最后加上下列代码
export SCALA_HOME=/opt/scala-2.11.6
export PATH=${SCALA_HOME}/bin:$PATH
检查是否安装成功
$ scala -version
成功则会显示
Scala code runner version 2.11.6 — Copyright 2002-2013, LAMP/EPFL

三.安装git

sudo apt-get install git

四.最后安装spark

我们到官网直接下载意见编译好的spark安装包

将下载好的文件解压
$ sudo tar xvf spark-1.3.1-bin-hadoop2.6.tgz -C /opt
添加环境变量
$ gedit /etc/profile
export SPARK_HOME=/opt/spark-1.3.1-bin-hadoop2.6

之后重启电脑，是/etc/profile永久生效

测试安装结果

1.打开终端，切换到Spark根目录

2.运行测试程序
$ ./bin/run-example SparkPi 10
最后得到正确结果

在Mac OS下给python安装模块

By Sun Qian

发表于 2015-06-02 1:58

今天晚上在用python实现粒子群算法的时候要用到numpy模块，奈何这不是系统自带的模块，便需要自己进行安装，折腾了很久，感觉网上不少方法都有问题，最后终于成功。于是自己决定整理一些可行的方法，希望能够帮到下次再遇到这些问题的新手。
方法1：通过工具自动化下载安装
easy_install,pip等工具可以自动帮你下载源码，并安装。而且mac系统已经默认集成了这几个工具。
所以直接在终端里输入参数即可：
sudo easy_install numpy
Mac下在命令前加上sudo来获得权限。
方法2:下载源码，手动运行setup.py install去安装
找到需要的模块源码，下载下来。往往都是.tar.ge, .zip后缀的压缩包，然后cd进入解压好的文件夹，运行 setup.py 即可。由于mac的下载路径是固定的，所以我们直接cd到下载路径内文件夹然后运行 setup.py
cd /Users/你的用户名/Desktop/模块文件夹名
python setup.py install
方法3:直接将文件拷贝到python的库目录
由于mac系统没有分区这一说，直接找很难找到目录。
我们打开finder，然后按 command+shift+G
即可，然后输入python的路径/Library/Frameworks/Python.framework/Versions/2.7/lib 然后将安装好的模块文件拷贝进去即去

记考试

By Sun Qian

发表于 2015-05-29 1:50

今天上午，本学期第一次正式的期末考试《信息论与编码》终于考完了。拿到试卷粗略看了一眼，放心不少，一共6题，和答疑时老师说的一样。题目也比较简单，估计大部分人都能上85+，只能说李发根真是个好老师。

转眼间，期末考试月就要来了，下个月貌似有四五门期末考，目测都不会有这次这么简单。而这次信息论与编码的考试，这门课应该是我这学期听过最多的一门课了吧，只有最后一章没有听过，复习时一共花了十几个小时把书翻了一遍，理解起来感觉也挺简单的，但记忆中上课时听得没这么简单，说明上课听讲还是很有作用的呀。

之前一直觉得，大学里的期末考试，考前都要把书看一遍，平时学了并没什么卵用，考前看下就可以了。可我现在貌似发现，虽然同样是看书，如果平时上课没有听过，期末看时得自己自学理解，效率会比上课听过后再看书自学复习要低很多很多！可惜真的感觉上大学后丧失听课能力了，总是开学初期会认真听几节课，之后就坚持不下去了，要么自己看书ppt自学，要么干醋玩儿或者做其他事情。想想剩下的四五门课，貌似都只是断断续续地听了一点，就感觉压力山大。再想想自己的gpa，自己保研或者出国的想法，感觉接下来一刻都不能浪费了，得拼命地看书预习呀！而其他还有一些未完成的事情也得安排时间做好！

大二还剩最后一个多月的时间，加油吧！

参加院友交流会的一些感悟

By Sun Qian

发表于 2015-05-23 3:25

今天下午（其实应该是昨天），我去参加了一个院友交流会。有4个2001级的院友在毕业十年之际，返回我们学校，同时也与我们这些学弟学妹分享一些个人的经历与经验。

就业

今天下午讨论的最多的话题就是就业。他们四个人目前几乎从事都是不同类型的工作，有两个学长在外企工作，仍然从事技术相关的售后工作；还有一个学长却在重庆英国领事馆从事英语培训方面的教育工作；另一个学长却是在一家创业公司呆了10年，经历了从一个员工数为个位的小公司到一个现在在全国多地均有分公司的大公司，目前已是副总。

可以说和他们交流了一下午，感觉我对自己以后的发展方向完全有了一个不同认识。

我一直有一个本科毕业后出国留学的打算，但觉得申请研究生的话毕业后也只能当码农，同时还要花费父母高额的学费，而且现在国内国外的研究生都一大堆，感觉需要读个博士才能有很大的竞争力，就有申请phd的想法。但今天听了几位学长的分享，发现他们几乎都是做着自己适合做的工作，他们也都强调要做适合自己，符合自己性格的工作；不一定非要热爱这份工作，但一定要适合做它。听他们这么一讲，我发现自己根本不适合读phd。我性格活跃，喜欢与人打交道，最关键的一点是我很难静下心来做事，根本不适合潜下心来搞科研，如果我申请到了phd，我也没有想要当什么科学家做什么重大研究，纯粹只是想混个高学历，提升自己的竞争力。但想想自己要花至少5年时间去做不喜欢不适合自己的事情，想必一定非常痛苦。貌似还不如读个两年研究生去工作。在公司里呆个3年对我的锻炼绝对比在实验室里呆3年要大的多。所以，也许还是读个研究生，积累一些工作经验，转到管理层或是去创业更适合我吧。

感觉自己文笔和排版实在太渣了，要多看看别人的博客，好好学习学习！

重新认识自己

By Sun Qian

发表于 2015-05-21 0:12

为什么突然想写博客呢？

－这学期深感自己在技术上欠的太多了，想用博客来记录自己每天看书的一些心得体会，记录自己的成长之路。
－想坚持做一件，每天写一些东西，每天坚持安静地坚持思考一会儿，反思自己。
－希望可以培养自己持续做一件事情的能力和自己渣到极限的写作能力。

第一篇博客，我想重新认识自己。很久没有跟自己对话了，想从九个我过去要回避的问题开始：

1、请你介绍一下自己，你是个什么样的人？

上了大学之后，自己经历了不少事情，变得极度理性。有明确的目标，知道自己该干什么，但却又是懒癌加身，总是做不到。对他人要求严格，对自己却各种找借口。性格外向乐观，总给人感觉是个大逗比。但只有我自己知道，我只是表现得很傻很逗很可爱，自己的内心还是挺成熟的。性格如此，没有办法。

2、你的理想是什么？这个理想是怎么形成的？

我的理想就是想当一个可以强大到影响社会国家的人，让国家更加发达。
可以用一句古话来说：为天地立心，为生民立命，为往圣继绝学，为万世开太平
初三之前，我一直浑浑噩噩地混日子，整天只知道尽最大可能去逃避学习而玩耍，直到有一天我看了一本书——《英雄志》。
英雄志里面有个主人公叫卢云，他出场时只是个落第书生，遭人迫害进了监狱，可在监狱里他仍不忘自己作为一个读书人的理想与责任，对当时看守他的狱卒说了这样的话“我辈读书之人，只求能为天地立心、为生民立命、为往圣继绝学、为万世开太平！生平全此四事，虽死无憾！”。当时我看了很是震惊，后来书中对卢云的一些描写，比如他只认定心中正确的道理，做对的事，坚持正道，影响了我。同时也引发了我对人生，对社会，国家的思考。从那时起，我便想当一个像卢云一样的人，为天地立心，为人民树立正确的价值观，让国家更加富强民主。不过现在我也知道这个想法很幼稚，当务之急是好好学习，以后能找一个好工作，再之后才能想这些事情。

3、你理想的伴侣关系是什么样的？你自己在这个伴侣关系中扮演什么样的角色？

谈起这个问题感觉很悲哀，目前还没有谈过恋爱，真是心疼我自己。我理想的伴侣关系应该是双方互相理解互相信任，可以携手并进，共同前行的，同时又是对方努力的动力以及督促者。在这个关系中，我希望自己可以起一个引导者的角色，可以带领和影响对方。

4、你理想的事业是什么，你正在做的工作符合你的事业理想吗？这份工作对你的意义是什么？

我理想的事业是当官，修身齐家治国平天下。现在还在上学，算是符合吧。目前就是提升自己，积累知识和经历的一个阶段，提升自己的专业知识，综合能力，扩展自己的视野。不过我现在学的是计算机，以后可能会当程序员，但程序员也可以通过自己写代码来改变世界，嘿嘿！

5、你对亲子关系怎么看？对你来说，什么是一个理想的父亲（母亲），你期望自己成为这样一个理想父亲（母亲）吗？

我觉得亲子关系很重要，父母应该是对孩子一生影响最大的人了吧，可以说父母的品行所作所为基本决定了孩子的素养和品质。我觉得一个理想的父亲应该是和孩子像朋友一样相处，给予自己的关爱，但同时要求得严格，但和孩子交流时却应该平等对待，像对朋友一样。同时要引导孩子去思考，观察这个世界，使之形成正确的三观。

6、你对钱怎么看，你认为赚到多少钱是足够的？如果你明天一早醒来，已经有足够的钱，你将会如何继续安排自己的生活？

钱，很重要！现在这个社会，没有钱真的什么都做不成。我认为赚到我想买个什么东西就可以去买，想吃什么就可以去吃，想去哪儿玩就能去哪儿，同时家里亲人生病时可以不考虑医药费这个问题差不多就算是足够的了吧。如果我明天一早醒来，已经有足够的钱，我还会继续好好上学，但会拿出部分钱投资自己，比如寒暑假出国学习长见识啊，人生在这个阶段必须努力，有些东西钱是买不到的。同时我会把钱拿去做投资，再生更多的钱。不会再花父母的钱，花钱给父母买各种礼物，带他们出去玩！

8、你的择友标准是什么？什么样的人你会愿意交往，什么样的人你会拒绝和他交往？

标准是多重的，而且有些人就是想交往，有些人就是讨厌，有时谈不上什么标准，应该还是接触后的一些感觉吧。我比较喜欢脾气好，性格好，开朗一些的人。聪明、优秀的人也是我愿意结交的对象。但是也不一定要求你有多优秀，最重要的还是人品好，尊重别人。我不是很喜欢势力，自私的人，有些人可能刚开始接触看不出来，但处久了总能发现的。总之，一定要和自己的口味才行呀。

9、你对死亡怎么看？你希望自己活到多少岁，你准备怎么度过从现在到死亡的这段时间？如果你要立遗嘱，这份遗嘱会怎么写？

我觉得死亡不可避免，而且人死了之后，就什么都没有了，从这个世界上彻底消失了！所以在死之前，我们一定不要虚度这一生！一定要做出点成就，活得精彩一些，对社会对他人做出点些许贡献！当然希望自己活得越久越好啦，最好能活到一百多岁！我想少浪费点时间，在死之前尽可能得多做一些有意义的事情！至于遗嘱，我现在不清楚会写多少，但肯定会有一点：1、不搞任何繁琐的仪式，直接火化然后可以海葬什么的

以上的这九个问题摘自《很少人能顺畅回答这9个问题——心理治疗刚开始医生常常会先问你的 》by 李孟潮。
