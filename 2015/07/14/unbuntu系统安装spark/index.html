unbuntu系统安装spark

一.安装java环境

$ sudo apt-add-repository ppa:webupd8team/java
$ sudo apt-get update
$ sudo apt-get install oracle-java7-installer

检查是否安装成功：
$ java -version

成功的话将会显示
java version “1.7.072” Java(TM) SE Runtime Environment (build 1.7.072-b14) Java HotSpot(TM) 64-Bit Server VM (build 24.72-b04, mixed mode)

二.安装scala

从网上下载安装包，可以直接从官网下载或从终端输入下列代码
$ wget http://www.scala-lang.org/files/archive/scala-2.11.6.tgz
$ sudo tar xvf scala-2.11.6tgz -C /opt #将scala压缩至 /opt/scala-2.11.6
设置环境变量
$ gedit /etc/profile
在打开的文件最后加上下列代码
export SCALA_HOME=/opt/scala-2.11.6
export PATH=${SCALA_HOME}/bin:$PATH
检查是否安装成功
$ scala -version
成功则会显示
Scala code runner version 2.11.6 — Copyright 2002-2013, LAMP/EPFL

三.安装git

sudo apt-get install git

四.最后安装spark

我们到官网直接下载意见编译好的spark安装包

将下载好的文件解压
$ sudo tar xvf spark-1.3.1-bin-hadoop2.6.tgz -C /opt
添加环境变量
$ gedit /etc/profile
export SPARK_HOME=/opt/spark-1.3.1-bin-hadoop2.6

之后重启电脑，是/etc/profile永久生效

测试安装结果

1.打开终端，切换到Spark根目录

2.运行测试程序
$ ./bin/run-example SparkPi 10
最后得到正确结果
